# 导演讲戏本

## P01 牙口不错

- 人物：霍宴、陈一一
- 场景：面试办公室
- 镜头组：多镜头（3 组）
- 时长建议：10s
- 镜头规划：极特写（肩膀） → 中景（霍宴） → 全景（对峙）
           平视 → 俯视 → 平视
           固定 → 缓拉 → 固定

**导演阐述**：
开场第一秒，镜头死死怼在霍宴的右肩上。深灰色西装被拉开一角，白衬衫领口下方的肩膀处贴着一条刺眼的透明创可贴，隐约透着暗红色的咬痕。随着镜头缓慢拉开，金丝眼镜后的霍宴显露全貌，他靠在高级皮椅上，指尖有节奏地敲击桌面，发出的闷响在安静的室内异常清晰。他嘴角勾起一抹似有若无的弧度，眼神像猎人锁定猎物般死死锁住对面的陈一一，轻笑一声：“陈小姐，牙口不错。”镜头切到陈一一，她身体僵硬，瞳孔在极短的时间内剧烈收缩，呼吸屏住，双手在膝盖上死死绞在一起。光影采用高对比的暖金顶光，营造出一种华丽的压迫感。

## P02 凌晨的羞辱

- 人物：陈一一、张建国
- 场景：律师事务所办公室
- 镜头组：多镜头（3 组）
- 时长建议：15s
- 镜头规划：全景建立 → 中景（冲突） → 特写（文件）
           平视 → 侧拍 → 俯冲
           固定 → 摇镜 → 固定

**导演阐述**：
画面闪白进入回忆。凌晨的律所，只有陈一一的工位亮着一盏孤零零的暖黄台灯，四周是死寂的冷蓝黑影。陈一一趴在桌上，发丝凌乱，眼底有明显的青色。张建国推门而入，皮鞋撞击地面的声音粗暴无礼。他将一沓厚厚的文件狠狠砸在陈一一脸上，纸张如雪片般散落，其中一张划过她的脸颊。陈一一惊醒，眼神从迷茫瞬间转为惊愕。镜头特写文件封面，张建国那三个字在灯光下显得格外讽刺。张建国肥腻的脸上满是冷笑，额头的油光反射着台灯的光点，他步步紧逼，陈一一不断后退，直到背部撞在冰冷的玻璃幕墙上。

## P03 冰美式的反击

- 人物：陈一一、张建国
- 场景：律师事务所办公室
- 镜头组：单镜头
- 时长建议：10s
- 镜头规划：中景 → 近景 → 随动
           平视 → 仰视 → 侧拍
           固定 → 快速推镜 → 摆动

**导演阐述**：
陈一一没有如预期般哭泣，她的眼神在瞬间变得极度冰冷。镜头对准她攥紧咖啡杯的手，指节因为用力而发白，杯壁上的水珠顺着指缝滑落。她猛地起身，动作干脆利落，右手划出一道弧线，深褐色的冰美式顺着张建国的地中海秃顶倾泻而下，液体顺着他油腻的脸颊和廉价衬衫流进领口。张建国整个人愣住，像被定格的滑稽雕塑。陈一一反手将辞职信重重拍在桌面上，发出一声清脆的闷响：“这脸留着给你当记事本吧。老娘不干了！”随后拎包转身，高跟鞋在空旷的走廊踏出决绝的节奏。

## P04 25 岁的微醺

- 人物：陈一一、霍宴
- 场景：酒吧、酒吧走廊
- 镜头组：多镜头（3 组）
- 时长建议：15s
- 镜头规划：特写（蛋糕） → 中景（踉跄） → 远景（相撞）
           俯视 → 平视 → 侧拍
           固定 → 缓跟 → 撞击感固定

**导演阐述**：
深夜酒吧，光影迷离。镜头先给到一个近景特写：吧台上放着一个精致却显孤独的小蛋糕，插着一根燃烧的蜡烛，火苗在杯盏交错的背景中微微晃动，映出“25”的字样。陈一一面色潮红，眼神涣散，她盯着火苗自嘲一笑，随后踉跄着走向洗手间。在走廊转角，她毫无防备地撞进一个宽阔硬挺的胸膛。镜头捕捉到霍宴西装面料的精细纹理和那股冷冽的雪松香气。霍宴原本冷漠的眼神在看清陈一一脸的瞬间骤然收缩，那是一种穿越八年时光的震颤。他的手指在身侧微微颤抖，原本想推开的手在半空停滞，最终化为虚扶在她腰间的克制动作。

## P05 肩膀上的印记

- 人物：陈一一、霍宴
- 场景：酒吧走廊
- 镜头组：多镜头（4 组）
- 时长建议：15s
- 镜头规划：特写（拉领带） → 极特写（咬肩膀） → 近景（掉落笔记本） → 极特写（内页文字）
           仰视 → 平视 → 俯视 → 垂直俯视
           固定 → 缓推 → 垂直下移 → 固定

**导演阐述**：
陈一一像只受委屈的小兽，她猛地扯住霍宴的深蓝色真丝领带，迫使他低下头。镜头极近距离捕捉她的动作：踮起脚尖，一口轻轻碰到他的肩膀上，随后狠狠咬下。霍宴发出一声沉闷的哼声，眉头微蹙，但双手却更紧地护住她的腰。随着两人的拉扯，一本泛黄的笔记本从霍宴西装口袋滑落，掉在暗红色地毯上。霍宴弯腰捡起笔记本，镜头切入极特写：笔记本内页纸张泛黄，有几行字迹因为长年累月的摩挲而变得模糊，只能隐约看见“法院……公道……”几个残缺的字。他低下头，在陈一一耳边极轻地说了一句：“终于找到你了，一一。”眼底翻涌着压抑了八年的执念。

## P06 八年前的账

- 人物：霍宴、陈一一
- 场景：天恒律所面试室
- 镜头组：多镜头（3 组）
- 时长建议：15s
- 镜头规划：全景（重逢） → 中景（逼近） → 特写（耳语）
           平视 → 随动 → 侧拍特写
           固定 → 缓推 → 环绕

**导演阐述**：
回到现实，面试室被明亮的自然光充盈，通透得近乎残酷。陈一一僵在门口，像被施了定身法。霍宴缓缓从主位起身，西装扣子扣得一丝不苟，唯独右肩处的衬衫隐约透出创可贴的轮廓。他一步步走近，皮鞋在地板上的每一步都像踩在陈一一的心跳上。他俯身凑到她耳边，用只有两人能听见的声音说：“陈小姐，昨晚弄脏我衬衫、弄皱我领带的账，我们慢慢算。”他顿了一下，声音低下去，带着一种危险的磁性：“更要算的，是八年前，你欠我的那笔账。”陈一一的表情从尴尬逐渐转为迷茫与惊恐，大脑中搜寻不到任何关于八年前的碎片。
